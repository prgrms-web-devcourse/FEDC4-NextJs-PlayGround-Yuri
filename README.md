## 진행 방식

1. 오픈 스크럼에 범위를 정한다.
2. 자신의 이름 폴더(`최원석`)에서 자유로운 형식으로 회차 목록에 해당하는 항목들을 스터디한다.
3. PR 규칙에 맞게 올린다.
4. 다음날 오픈 스크럼 때 `review` 후 `merge` 한다. (1부터 다시 반복)

## Github 규칙

### Branch 규칙

- 그 회차 및 이름 형식(`01-최원석`)으로 생성한다

### PR 규칙

1. 자신의 이름 폴더(`최원석`)에서 자유로운 형식으로 회차 목록에 해당하는 항목들을 스터디한다.
2. `[1회차] 최원석` 형식으로 PR 올린다.
